# Menu
- 메인 페이지
  ```
  추천공연, 오픈 티켓, 장르별 랭킹
  ```
  
- 일반 회원 서비스  
  ```
  회원가입(다음 도로명주소 API)/로그인/탈퇴, 회원 정보 관리, 아이디/비밀번호 찾기(이메일 전송), 
  장르별/지역별 공연 조회(공연장-카카오맵 API), 공연 예매(아임포트 API), 공연 기대평/관람평, 
  예매 목록 조회, 공연 찜하기, 최근 본 공연, 고객센터
  ```

  ```

  ```
